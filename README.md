# Auto increment tag

This action adds one more to the last number of a tag. Example: 1.23.1 to 1.23.2, 4.3 to 4.4, 5 to 6...

## Inputs

### `token`

Secret github token: ${{secrets.GITHUB_TOKEN}}

## Outputs

### `tag`

Tag created.

## Exemplo de uso

uses: sophiware/github-actions-auto-increment-tag@v1
